# csgo_discord

Discord bot to control lgsm 

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install python.valve
pip install config
pip install discord
```
## Requires
```bash
python 3.9
```
## Usage

```bash
Edit config_example.py and save as config.py
.help in discord to see commands
```
